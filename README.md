# Polymer 2 Application

## Requirements:-
`node > ^6.0.0`<br>
`npm > ^3.0.0`<br>
Bower: `npm install -g bower`<br>
Polymer: `npm install -g polymer-cli`<br>

## Running the Project
`bower install` <br>
`polymer serve`

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.
